8-Tab GUI Workflow

A professional WindPRO-inspired tabbed interface guiding the complete wind farm design workflow — from project setup through to final energy yield results. Each tab provides a focused workspace with context-sensitive tools and visualisations.

OpenStreetMap Terrain Visualization

High-resolution terrain data rendered on interactive basemaps via contextily. Supports multiple tile providers including OpenStreetMap, Stamen Terrain, and ESRI World Imagery for detailed topographic context.

contextily OpenStreetMap Rasterio Matplotlib

ESA CCI Land Cover 300m Roughness

Global land cover data from the European Space Agency Climate Change Initiative at 300m resolution, automatically converted to surface roughness lengths using standard WAsP land-use classification tables. Includes full WAsP .map file export for seamless interoperability.

ESA CCI 300m Resolution Roughness Length .map Export

Global Wind Atlas API Integration

Direct integration with the DTU Global Wind Atlas v2 API for wind climate data retrieval at any location worldwide. Implements robust POST request handling with intelligent fallback strategies for reliability across variable network conditions.

DTU GWA v2 REST API POST Method Fallback Logic

Interactive Map Area Selection

Draw custom polygons, rectangles, and circles directly on an interactive Leaflet map to define wind farm boundaries. Powered by Leaflet.Draw for intuitive area selection with coordinate export and GeoJSON support.

Leaflet Leaflet.Draw GeoJSON QWebEngineView

WAsP-like Wind Flow Model

Industry-standard linearised wind flow modelling implementing Jackson-Hunt theory for speed-up over hills, Internal Boundary Layer growth for roughness transitions, and the Envelope of Wind Atlases (EWA) method for complex terrain. Matches WAsP modelling approach for verified results.

Jackson-Hunt IBL Growth EWA Method NumPy/SciPy

Layout Optimization

Four optimisation algorithms for turbine placement — from simple regular grid generation to advanced metaheuristic approaches. Grid layout for initial scoping, Greedy with KD-tree for rapid placement, Particle Swarm Optimisation for multi-objective AEP maximisation, and Genetic Algorithms for constrained global optimisation.

Grid Layout Greedy / KD-tree PSO Genetic Algorithm SciPy Optimize

Wake Models + 12-Sector Weibull AEP

Three industry-standard wake models for deficit calculation — Jensen (top-hat), Frandsen (Gaussian-like), and Ainslie (eddy viscosity k-ε). Combined with 12-sector Weibull distribution fitting for direction-dependent Annual Energy Production calculation.

Jensen Frandsen Ainslie 12-Sector Weibull AEP Calculation

.map File Export

Full WAsP binary and text format .map file generation with proper roughness change records, elevation contours, and site calibration data. Includes automatic UTM reprojection from geographic coordinates to local projected coordinate systems for direct WAsP compatibility.

WAsP Binary WAsP Text UTM Reprojection pyproj

PyInstaller .exe Packaging

Distributed as a standalone Windows .exe with all dependencies bundled — no Python installation required. Custom PyInstaller spec file with UPX compression, icon embedding, and single-file mode for professional deployment. Includes automatic version stamping and digital signing readiness for enterprise distribution.
