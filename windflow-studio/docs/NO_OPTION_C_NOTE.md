# Option C Removed

This is the simplified production version before the experimental Option C changes.

Removed / not included:
- FFT / spectral terrain-flow solver
- WAsP benchmark calibration workflow
- calibration.json import/export
- sector calibration UI

Kept:
- GWA point wind climate download
- ERA5 / ERA5T time-series download
- terrain download and contour generation
- OSM roughness download
- WAsP-like vertical and horizontal extrapolation approximation
- upstream effective roughness approximation
- multi-scale directional terrain speed-up approximation
- Jensen/PARK wake model
- AEP and time-series reporting

Use this version if you want a simpler, easier-to-understand model without benchmark calibration steps.
