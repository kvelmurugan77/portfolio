3 · Terrain & roughness

Auto-downloads DEM (Open-Meteo / OpenTopo) and land-cover roughness (OSM). Uses paced API calls + cache to reduce HTTP 429. Use grid 25–30. If rate-limited, wait 1–2 min or re-use cache (same area). Roughness falls back to z₀ quickly if OSM is busy.

4 · Wind climate

Pick one primary climate. Site mast CSV is best. ERA5 = long-term reanalysis. GWA = microscale atlas (.lib). NEWA/other mesoscale via CSV if you have it.

Active wind in state: None (will be auto-loaded on run)
Drop wind CSV / GWA .lib or click

GWA from the browser is often blocked by CORS. Prefer Upload .lib (from globalwindatlas.info) or ERA5.
